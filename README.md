# homebrew-pikvm

Homebrew tap for [`pikvm`](https://github.com/j4y-w4lk3r/pikvm) — a terminal UI and CLI for PiKVM ATX power control.

## Install

```bash
brew install j4y-w4lk3r/pikvm/pikvm
```

## Update

```bash
brew update
brew upgrade pikvm
```

## What this tap is

This repository is **auto-managed** by [GoReleaser](https://goreleaser.com/) on every tagged release of [`j4y-w4lk3r/pikvm`](https://github.com/j4y-w4lk3r/pikvm). The `Formula/pikvm.rb` file is regenerated and pushed automatically — please don't edit it by hand.
